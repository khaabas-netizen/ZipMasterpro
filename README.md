##  Description

**ZipMaster Pro** is an advanced file compression and extraction tool for Windows, developed in C# using Windows Forms. It provides a user-friendly interface combined with professional features for efficient archive management.

## ✨ Key Features

### 🔐 Security & Protection
- **AES-256 Password Protection**: Secure your archives with strong AES-256 encryption.

### 📦 Archive Management
- **Archive Preview & Browse**: Preview and browse archive contents before extracting.
- **Recent Files History**: Quick access to your recently used files and archives.
- **Split Archive by Size**: Split large archives into smaller, manageable parts.

### 🔍 Advanced Tools
- **Duplicate File Detection**: Easily find and manage duplicate files.
- **System Tray Notifications**: Keep track of compression/extraction progress via system tray alerts.

### 📂 Supported Formats
- ZIP
- TAR
- GZ (GZip)
- BZ2 (BZip2)

### ️ System Integration
- **Context Menu Integration**: Seamlessly integrate ZipMaster Pro into your Windows right-click context menu.

## 🚀 Getting Started

### Prerequisites
- **OS**: Windows 7 or later
- **Framework**: .NET Framework 4.5 or higher

### Installation
1. Download the latest release of ZipMaster Pro.
2. Run the `ZipMaster Pro.exe` executable.
3. *(Optional)* Use the "Add to Right-Click" feature in the System menu to integrate it with Windows Explorer.

### How to Use
1. Click **Add Files** or **Add Folder** to select the items you want to compress.
2. Choose your desired compression level: **Fast**, **Optimal**, or **Max**.
3. Click the **Compress** button to start the process.

## 📋 Interface Overview

### File Actions
- **Add Files**: Add individual files to the queue.
- **Add Folder**: Add an entire folder to the queue.
- **Clear All**: Clear the current file list.

### Tools
- **Set Password**: Apply password protection to your archive.
- **Preview Archive**: View contents without extracting.
- **Recent Files**: Access your history.
- **Split Archive**: Divide archives by specific size limits.
- **Find Duplicates**: Scan for duplicate files.
- **Format**: Select the output format (ZIP, TAR, etc.).

### System
- **Add to Right-Click**: Add to Windows context menu.
- **Remove Right-Click**: Remove from Windows context menu.
- **Settings**: Configure application preferences.
- **About**: View version and developer information.
